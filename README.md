# demo-aws

A sample project for AWS demonstration with Express, TypeScript, and Prisma.

## Setup

```sh
npm install
```

## Development

```sh
npm run dev
```

## Build

```sh
npm run build
```

## Type Check

```sh
npm run typecheck
```

## Start

```sh
npm start
```

## Database

Reset Prisma migrations:

```sh
npm run db:erase
```
